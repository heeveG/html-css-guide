In this part of the tutorial we will take a look at a couple of examples of how Flexbox layout works


Stretch all, fixed spacing

In the example below you can see five blocks built with Flexbox layout.
There is a one parent container, which is styled with display set to flex
.container {
     display: flex;
}

and the child elements - five containers, that are styled like so
.item {
     flex-grow: 1;
    height: 100px;
}

The main style settings to look at are display and flex-grow. Display sets the layout method for child elements of the container and flex-grow will increase the amount of space that an element is allowed to stretch to compared to any other element.
You can experiment with it by stretching and shrinking the window.


Stretch middle, fixed spacing

This one is an extended version of the previous example. The same parent container is used, however, child elements are now styled differently
.item-2 {
    height: 100px;
    width: 100px;
}

.item-middle {
    flex-grow: 1;
}

Now only certain elements have the flex-grow setting, while others remain unchanged, when scaling



3x3 Grid

Now we will create a 3x3 grid with Flexbox layout method.

The CSS of the parent container
.container-2 {
    display: flex;
    flex-wrap: wrap;
    justify-content: space-between;
}
The child containers
.item-3 {
    flex: 0 32%;
    height: 100px;
    margin-bottom: 2%;
}
The result

Same as before, containers are perfectly scalable
