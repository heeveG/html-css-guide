In this part of the tutorial we will take a look at a couple of examples of how Grid layout works


Defining a Grid

To define a Grid use display:grid or display:inline-grid on the parent element. You can then create a grid using the grid-template-columns and grid-template-rows properties
.grid-container {
    display: grid;
    grid-template-columns: 200px 200px 200px;
    grid-template-rows: 100px 100px;
    grid-gap: 10px;
}
We have just created a grid that consists of three columns of width 200px and two rows of height 100px.
Now, to create grid items - children of out grid-container, we don't actually need to specify any grid-specific settings - just style them as usual containers
.grid-child {
    padding: 20px;
}
All direct children of the parent now become grid items and the auto-placement algorithm lays them out, one in each grid cell. Creating extra rows as needed
Just like that, we have built our first Grid container

Defining a Grid with custom areas

In this example we are going to build a Grid layout, that is going to contain grid cells of size bigger than default cell. You can see how that looks at the bottom.

Here is the code for the parent container
.grid-container-2 {
    display: grid;
    grid-gap: 10px;
    grid-template-columns: 200px 200px 200px;
    grid-template-rows: 100px 100px;
}
Now, we are going to define a styling for each of child elements separately
.grid-child-2 {
     padding: 20px;
}
.a {
     grid-column: 1 / 3;
     grid-row: 1;
}
.b {
     grid-column: 3 ;
     grid-row: 1 / 3;
}
.c {
     grid-column: 1 ;
     grid-row: 2 ;
}
.d {
     grid-column: 2;
     grid-row: 2;
}
Observe, how we distribute the space of the grid between five child elements (a-d)

C
D
Thus, we have created out custom grid.
