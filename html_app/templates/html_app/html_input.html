Input


This section is about input element. It must be obvious where the input element is used in out form.
But just to make sure - inspect the Element code and you'll see that the form contains three input sections:
1. Name
2. E-mail
3. And last one is a 'Send your message button'
Each one of those input elements contains type attribute, that defines the type of input it will accept.
The Name input's type is text - which means simple text area.
The E-mail's type is email, and it's there for a reason. It applies default validations to that field. You can try to type in an invalid email and see what happens.
Last one is a sumbit type input - which means that it's a some sort of button that is submitting our form. It also has a value attribute, which is the text inside of the button.
The message field could've been an input too, but I wanted to show you that the there is a element textarea for bigger text fields.
Each input element also has a name attribute, which is used to reference elements in a JavaScript, or to reference form data after a form is submitted.
They also contain an id attribute, which is self-explanatory.
