Anchor



If you haven't yet noticed, the Message field is not only different from the other ones by being textarea element, but it also contains an anchor.

When you click it, it actually redirects you to the /html/anchor/ page, which, in fact, changes nothing, but is a good example to show my point.

Anchors usually have href attribute, which points to the link to be redirected to.

They also have target attribute, which might be helpful sometimes. It specifies where to open the linked document.
Another useful attribute is title. It specifies extra information about an element.
Here is an example - Hover over me

Anchors are easy, but they are very crucial part, when building any website.
