Unordered list



By now, you know almost everything about this form. However, I never told you about its structure.

You can see in the Element code, that the very first element after the form element is ul element.
It defines a structure of the form - each input box lies within one li element. Such structure is very easy-to-read and easy-to-style one.

Now, I have been talking a lot about styling things, before you move to the next chapter about CSS (which might be more intermediate level), you must know how styling works. Each HTML tag contains the class attribute, which is used when styling your page in CSS.
You can observe that out form has class myform, which means we could style it by writing:
.myform { } in the css file.

You can style elements by id and by the tags too!
